ℹ️ About Agentic Project Management (APM)

What is APM?

APM is an open-source framework designed for managing ambitious software projects with AI assistants. Instead of using a single, increasingly chaotic chat context, APM coordinates specialized AI agents (Planner, Manager, and Workers) working as a cohesive team to execute tasks efficiently.

📖 How to Use

  1. Define Workspace Scope: Navigate to Workspace Files in the sidebar. Describe your milestone under the Milestone tab, and list your development team under the Employees tab (as a JSON array).
  2. Set LLM Credentials: Click LLM Settings in the sidebar. Choose your API provider (Gemini, Anthropic, or OpenAI), set your API key, and specify any custom model overrides.
  3. Generate & Decompose: Tapping Decompose Milestone triggers the Task Generation Agent. The LLM breaks down the milestone, figures out task dependencies, schedules dates within the milestone window, and assigns tasks to the best-suited employees.
  4. Visualize & Refine: Use the Timeline (Gantt chart), Kanban, List Table, and Analytics views to inspect schedules and workload balance. Click on any task card or row to edit its attributes.
  5. Save Results: Click the Save tasks.json CTA in the top right to write the compiled schedule directly back to your local workspace files.

🤖 Important Things About the Models

  • Google Gemini (Recommended): Selected by default. Highly cost-effective and extremely fast. Adheres precisely to the required JSON schema output structure.
  • Anthropic Claude: Unparalleled reasoning capabilities for logical decomposition and ordering complex dependency tracks.
  • OpenAI GPT: Solid, consistent task scheduling heuristics and reliable JSON-object structure rendering.
  • Anti-Hallucination Boundaries: APM enforces strict validators to ensure that dates remain within milestone windows, task priorities map correctly, and tasks are assigned only to resources listed in your employee JSON.
